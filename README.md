## Setup

1.  Install a recent version of `Node.js` (>=10):

    ```
    brew install nodejs
    ```

1.  Install Cocoon:

    ```
    npm install -g @cocoon/cocoon
    ```

## Usage

Run the CLI via `cocoon` and the editor via `cocoon-editor`.
